# MedDiab Learn

MedDiab Learn is an interactive patient case simulation platform designed to help users practice clinical reasoning and diabetes treatment decision-making through guided learning scenarios.

## Features
- Interactive patient case simulations
- Guided clinical reasoning workflow
- Diabetes treatment decision-making
- Educational learning scenarios
- User-focused healthcare education

## Technologies Used
- HTML
- CSS
- JavaScript

## My Role
Collaborated in a hackathon team to help design and develop the interactive learning experience and patient case workflow.
